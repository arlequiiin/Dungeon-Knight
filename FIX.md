# Dungeon Knight - Known Bugs


1. Камера не следует за игроком в лобби
- **Файл:** `Assets/Scripts/PlayerController.cs` -> `OnStartLocalPlayer()`
- **Суть:** `SetTarget()` вызывается в `OnStartLocalPlayer()`, который срабатывает один раз. При смене сцены камера на новой сцене не привязывается к игроку.
- **Исправление:** привязывать камеру при каждой смене сцены или в `OnEnable()`.

2. Игрок не может атаковать, когда выбирает персонажа в лобби

3. Мобы не могут двигаться когда атакуют

4. Реализовать меню игры и проверить возможность подключения по локальной сети

5. Спрайты неправильно сортируются относительно своей позиции, хотя слой везде один и pivot установлен верно (сортировка по нему)

6. Исправление системы комнат - мобы появляются только когда игрок зашёл в неё. Пока игрок не прошёл комнату, он не может из неё выйти (мобы тоже не могут выйти за пределы комнаты). Реализовать несколько "волн" противников. 

7. Добавить эффект уклонения для героев
