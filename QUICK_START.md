# Быстрый старт: Процедурная генерация подземелья

## 🚀 Самый быстрый способ (1-2 минуты)

### Способ 1: Автоматическая генерация (рекомендуется для начинающих)

1. **Открой Unity проект**

2. **В главном меню выбери:**
   - `Dungeon/Setup/Create Default Tiles` → создаст простые тайлы для тестирования
   - `Dungeon/Setup/Create Test Scene` → создаст готовую сцену с Grid, Tilemap и Generator

3. **Нажми Play** → подземелье сгенерируется автоматически

4. **Смотри консоль** → там выведется информация о структуре подземелья

---

## 📋 Способ 2: Ручная настройка (более гибкий)

### Шаг 1: Подготовка Grid и Tilemap

```
1. Right Click в иерархии
2. 2D Object → Tilemap → Rectangular
3. Переименуй на "DungeonTilemap"
```

**Результат:** Grid с одним Tilemap на сцене

### Шаг 2: Создание тайлов

**Вариант A - Если есть текстуры:**
```
1. Создай папку Assets/Tiles
2. Перетащи туда текстуры
3. Выбери текстуру → Inspector → Sprite Mode: Multiple → Slice
4. Right Click в Tiles → Create → Tile
5. Создай FloorTile и WallTile с нужными спрайтами
```

**Вариант B - Быстро без текстур:**
```
1. Меню → Dungeon/Setup/Create Default Tiles
2. Готово!
```

### Шаг 3: Создание DungeonConfig

```
1. Right Click в Assets → Create → Scriptable Object → DungeonConfig
2. Назови "DungeonSettings"
3. Установи параметры:
   - Map Width: 100
   - Map Height: 100
   - Normal Rooms Count: 15
   - Special Rooms Count: 2
   - Min Path To Boss: 5
   - Max Path To Boss: 10
```

### Шаг 4: Создание Generator GameObject

```
1. Right Click в иерархии → Create Empty
2. Назови "DungeonGenerator"
3. Add Component → Generator
4. Перетащи DungeonSettings в поле "Dungeon Config"
5. Поставь галочку "Generate On Start"
6. Поставь галочку "Debug Mode"
```

### Шаг 5: Настройка DungeonRenderer

```
1. Выбери "DungeonTilemap" в иерархии
2. Add Component → DungeonRenderer
3. Перетащи WallTile в поле "Wall Tile"
4. Перетащи FloorTile в поле "Floor Tile"
```

### Шаг 6: Связь компонентов

```
1. Выбери "DungeonGenerator"
2. В компоненте Generator → поле "Dungeon Renderer"
3. Перетащи "DungeonTilemap" из иерархии
```

### Шаг 7: Запуск

```
Нажми Play ▶
```

## 🎮 Интерактивная регенерация

Во время Play'а можно пересоздавать подземелье:

1. Выбери "DungeonGenerator" в иерархии
2. В Inspector нажми кнопку "Regenerate Dungeon"
3. Подземелье пересоздастся с другим seed'ом

## ⚙️ Параметры DungeonConfig

```csharp
// Размеры карты в тайлах
mapWidth = 100          // Ширина карты
mapHeight = 100         // Высота карты

// Комнаты
normalRoomsCount = 15   // Количество обычных комнат с мобами
specialRoomsCount = 2   // Количество специальных комнат (магазины)

minRoomSize = (8, 8)    // Минимальный размер комнаты
maxRoomSize = (20, 20)  // Максимальный размер комнаты

// Граф и маршруты
minPathToBoss = 5       // Минимальное расстояние до босса
maxPathToBoss = 10      // Максимальное расстояние до босса
branchProbability = 0.3 // Вероятность боковых веток (0.0 - 1.0)

// Генерация
useRandomSeed = true    // Использовать случайный seed
seed = 0                // Фиксированный seed (если useRandomSeed = false)
```

## 🐛 Частые проблемы

| Проблема | Решение |
|----------|---------|
| Комнаты не видны | Проверь, что Floor Tile назначен и Tilemap видим |
| Ошибка компилации | Убедись, что все файлы в папке Generator |
| Нет логов в консоли | Проверь, что Debug Mode включён в Generator |
| Слишком мало/много комнат | Измени normalRoomsCount в DungeonConfig |
| Комнаты очень малы/велики | Измени minRoomSize и maxRoomSize |

## 📚 Архитектура

```
Generator.cs                ← Main MonoBehaviour (запускает процесс)
    ↓
DungeonConfig              ← Параметры генерации
    ↓
BSPGenerator               ← Основная логика (BSP + Graph)
    ├→ BSPNode            ← Дерево разбиения пространства
    ├→ Room               ← Структура комнаты
    └→ RoomType           ← Enum типов комнат
    ↓
DungeonRenderer            ← Визуализация на Tilemap
    ├→ Tilemap            ← Unity компонент
    ├→ FloorTile          ← Спрайт пола
    └→ WallTile           ← Спрайт стены
```

## 🎯 Следующие шаги

После того как всё работает:

1. **Коридоры** - добавить физические коридоры между комнатами
2. **Враги** - спаун мобов в зависимости от типа комнаты
3. **Ловушки** - элементы в специальных комнатах
4. **Предметы** - сундуки и лут в комнатах
5. **Визуальная полировка** - разные тайлы для разных типов комнат

## 💡 Советы

- **Используй один seed для тестирования:** отключи `useRandomSeed` и установи фиксированный `seed`
- **Debug режим замедляет генерацию:** отключи его когда закончишь отлаживать
- **Экспериментируй с параметрами:** попробуй разные комбинации для получения интересной геометрии
- **Проверяй консоль:** там выводится вся полезная информация о сгенерированном подземелье
